CosmoNV2 Documentacao

Rodando o projeto no Windows

  1. Fork nosso repositório.
  2. Use o Gitbash para clonar o repositorio: $ git clone https://github.com/seu_usuario/Documentacao.git
  3. Usando seu terminal acesse a pasta Documentacao: $ cd Documentacao
  4. Instale as dependências(use um ambiente virtual): $ pip3 install -r requirements.txt

  5. Para rodar o servidor, certifique-se de que esteja diretorio Documentacao.

    $ mkdocs serve

  6. acesse o site aqui, ele ficara hospedado localmente em http://127.0.0.1:8000/.

Rodando o projeto Linux(distros baseados em debian)

  1. Fork nosso repositório, ele aparecar como um clone no seu github
  2. Use Terminal para clonar o repositorio (precisa que o git esteja instalado): $ git clone https://github.com/seu_usuario/Documentacao.git
  3. Usando seu terminal acesse a pasta Documentacao: $ cd Documentacao
  4. Instale as dependências(use um ambiente virtual): $ pip3 install -r requirements.txt

  5. Para rodar o servidor, certifique-se de que esteja diretorio Documentacao.

    $ mkdocs serve

  6. acesse o site aqui, ele ficara hospedado localmente em http://127.0.0.1:8000/.
