# Alto Falante

`Altofalante.Altofalante`: Método Construtor

`Altofalante.parar()`: Faz o alto falante parar de tocar

`parâmetros`:
 - qu`uint8_t antidade_de_vezes``: Quandtidade de vezes que o alto falante precisa bipar
